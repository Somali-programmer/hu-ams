# Run and deploy HU-AMS

This contains everything you need to run your app locally.

View this App : https://hu-final-year-project-front.vercel.app

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the  in [.env.local](.env.local) to your  API key
3. Run the app:
   `npm run dev`
